Maximumus — standalone (только браузер)
========================================

Как открыть
  Дважды кликни по index.html или перетащи в браузер.
  Работает без интернета и без сервера (file://).

Управление
  Мышка — таяние льда.
  [H] — панель настроек справа (по умолчанию скрыта).

Текущая сборка
  Картинки уже встроены в index.html (data URL). Достаточно одного файла index.html —
  открой его по file:// и всё работает. Можно скинуть только index.html.

Пересобрать (если менял картинки)
  В этой папке выполни:  node build-embed.js
  Скрипт прочитает nature.jpg, n1.png, prompt1.png и т.д. и вставит их в index.html.

Скинуть людям
  Вариант 1: только index.html (в нём уже всё встроено).
  Вариант 2: папку maximumus-standalone в zip — пусть откроют index.html.
